Number Systems

A visual guide for high school — every set lives inside a bigger one

ℝ Real Numbers
Every number you can place on a number line
Irrational Numbers
Cannot be written as a fraction p/q
  • π = 3.14159265… (never repeats)
  • √2 = 1.41421356… (never repeats)
  • √3 = 1.73205080…
  • e = 2.71828182… (Euler's number)
  • √5, √7, √11 …
Decimal goes on forever with no repeating pattern
ℚ Rational Numbers
Can be written as p/q where p,q are integers, q≠0
ℤ Integers
…−3, −2, −1, 0, 1, 2, 3…
ℕ Natural Numbers
Counting numbers: 1, 2, 3, 4, 5, …
Sometimes 0 is included (depends on convention).
These are the numbers we first learn to count with.
Zero & Negative Integers
0 — neither positive nor negative
−1, −2, −3, … — opposite of natural numbers
Fractions & Terminating / Repeating Decimals
½ = 0.5  ·  ¾ = 0.75  ·  ⅓ = 0.3  ·  ⅔ = 0.6  ·  4/3 = 1.3
Decimal either stops (terminates) or repeats in a pattern — so it IS rational.
⬆ Everything above lives on the Real Number line

∞ Repeating Decimals (Rational!)
A bar over digits means they repeat forever — but because the pattern repeats, these ARE rational (they equal a fraction).
1/3 = 0.3 = 0.3333…
2/3 = 0.6 = 0.6666…
4/3 = 1.3 = 1.3333…
1/7 = 0.142857
5/6 = 0.83 = 0.8333…
1/11 = 0.09 = 0.0909…
Trick: to convert 0.3 → fraction: let x = 0.333…, then 10x = 3.333…, subtract: 9x = 3, so x = 3/9 = 1/3 ✔
