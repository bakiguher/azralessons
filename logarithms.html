Logarithms

The inverse of exponentiation — "which power gives this number?"

💡 The Core Idea

A logarithm answers the question: "to what power must I raise the base to get this number?"

logb(x) = y ⟺ by = x
These two statements mean exactly the same thing — just written differently
b = base
Must be positive and ≠ 1. Usually 10, e, or 2.
x = argument
The number you want the log of. Must be > 0.
y = result
The exponent / power. Can be any real number.
📐 The Three Common Bases
log(x)
Common Log
Base 10 — when no base is written, assume 10.
Used in science & engineering.
log(100) = 2
because 10² = 100
ln(x)
Natural Log
Base e ≈ 2.718
Used in calculus & physics.
ln(e) = 1
because e¹ = e
log₂(x)
Binary Log
Base 2
Used in computer science.
log₂(8) = 3
because 2³ = 8
★ Special Values to Remember
Log of 1 is always 0
logb(1) = 0
because b⁰ = 1 always
Log of the base itself is 1
logb(b) = 1
because b¹ = b always
Log of a fraction < 1 is negative
log10(0.01) = −2
because 10⁻² = 0.01
Log of 0 or negative = undefined
logb(0) = undefined
no power of b ever reaches 0
📚 The 5 Logarithm Rules

These work for any base b > 0, b ≠ 1

1. Product Rule
logb(a · c) = logb(a) + logb(c)
log(2 · 50) = log(2) + log(50) = log(100) = 2
2. Quotient Rule
logb(a / c) = logb(a) − logb(c)
log(1000/10) = log(1000) − log(10) = 3 − 1 = 2
3. Power Rule
logb(aⁿ) = n · logb(a)
log(10³) = 3 · log(10) = 3 · 1 = 3
4. Change of Base
logb(x) = log(x) / log(b) = ln(x) / ln(b)
log₂(32) = log(32)/log(2) = 1.505/0.301 = 5
5. Inverse / Cancellation
blogb(x) = x   and   logb(bx) = x
10^log(7) = 7  ·  log(10⁵) = 5
🔎 How to Solve — The Method

Always convert log → exponent form first: logb(x) = y  →  by = x

Type 1: Find the result y
log3(81) = ?
→ Ask: 3 to what power = 81?
→ 3¹=3, 3²=9, 3³=27, 3⁴=81
→ Answer: y = 4
Type 2: Find the argument x
log2(x) = 5
→ Convert: 2⁵ = x
→ x = 32
Type 3: Find the base b
logb(125) = 3
→ Convert: b³ = 125
→ b = ∛125 = 5
✎ Worked Examples
Example 1  ·  log₁₀(1000) = ?
1
Ask: 10 to what power = 1000?
2
10¹ = 10  ·  10² = 100  ·  10³ = 1000 ✔
log(1000) = 3
Example 2  ·  log₂(1/8) = ?
1
Ask: 2 to what power = 1/8?
2
1/8 = 2⁻³  (since 2³ = 8, so 2⁻³ = 1/8)
log₂(1/8) = −3
Example 3  ·  log₅(x) = 3 → find x
1
Convert to exponent: 5³ = x
2
5³ = 5 × 5 × 5 = 125
x = 125
Example 4  ·  log(x) + log(x+3) = 1  (base 10)
1
Product rule: log(x · (x+3)) = 1
2
Convert: x(x+3) = 10¹ = 10
3
x² + 3x − 10 = 0  →  (x+5)(x−2) = 0
4
x = −5 or x = 2  ·  but log(negative) is undefined, so x = −5 is rejected
x = 2
Example 5  ·  log₄(64) using change of base
1
Change of base: log(64) / log(4)
2
log(64) = log(2⁶) = 6·log(2)  ·  log(4) = log(2²) = 2·log(2)
3
Result: 6·log(2) / 2·log(2) = 6/2
log₄(64) = 3   (check: 4³ = 64 ✔)

🌎 Where Logarithms Appear in Real Life
🎵 Decibels (sound)
dB = 10 · log(I / I₀)
A sound 10× louder = +10 dB
🌋 Richter Scale (earthquakes)
Each +1 on the scale = 10× more energy
⚖ pH (acid/base chemistry)
pH = −log[H⁺]
pH 4 is 10× more acidic than pH 5
📈 Compound Interest
t = log(A/P) / log(1+r)
Find how many years to reach amount A
