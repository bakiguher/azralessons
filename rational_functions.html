Part 1 — Rational Functions

1. Definition

A rational function is a function that can be written as a fraction of two polynomials:

f(x) = p(x) / q(x) where p(x) and q(x) are polynomials and q(x) is not the zero polynomial
✔ Examples (rational)
f(x) = (2x² − 3x + 1) / (x⁵ − 4x³ + x − 10)
g(x) = 1/x
h(x) = x³ − 2x + 3  (= (x³−2x+3)/1, polynomial)
✘ Counter-example (NOT rational)
f(x) = (√x − 5) / (3√x + 6)
√x is not a polynomial term

2. Domain

dom f = ℝ \ { x | q(x) = 0 } Exclude every x where the denominator equals zero

Worked examples

f(x) = (x² + 4) / (x² − 9)
1
Set denominator = 0: x² − 9 = 0 → (x−3)(x+3) = 0
2
Solutions: x = 3 or x = −3
dom f = ℝ \ {−3, 3}
f(x) = (x² − 4) / (3x² − 5x − 2)
1
Factor denominator: 3x² − 5x − 2 = (3x + 1)(x − 2)
2
(3x + 1)(x − 2) = 0 → x = −⅓ or x = 2
dom f = ℝ \ {−⅓, 2}
f(x) = (5x+3)(x+1) / (2x³ − x² − 18x + 9)
1
Factor denominator by grouping: 2x³ − x² − 18x + 9 = x²(2x−1) − 9(2x−1) = (x²−9)(2x−1)
2
= (x−3)(x+3)(2x−1) = 0 → x = 3, x = −3, x = ½
dom f = ℝ \ {−3, ½, 3}

3. Zeros (Nulwaarden)

Zeros = values of x where numerator = 0 AND denominator ≠ 0 If both numerator AND denominator = 0 at the same x → it is a "hole", NOT a zero
f(x) = (x² + 7x + 12) / (x² − 25)
1
Factor numerator: x² + 7x + 12 = (x+3)(x+4) → zeros of numerator: x = −3 or x = −4
2
Factor denominator: x² − 25 = (x−5)(x+5) → poles: x = ±5
3
Neither −3 nor −4 makes the denominator zero ✔
Zeros: x = −3 and x = −4  ·  dom f = ℝ \ {−5, 5}
f(x) = (x² + 7x + 12) / (x² − 16) — careful! shared factor
1
Factor numerator: (x+3)(x+4) → candidates: x = −3 or x = −4
2
Factor denominator: (x−4)(x+4) → poles: x = 4 or x = −4
3
x = −4 makes BOTH numerator and denominator zero → hole, not a zero!
4
x = −3 makes denominator (−3−4)(−3+4) = (−7)(1) = −7 ≠ 0 ✔
Only zero: x = −3  ·  dom f = ℝ \ {−4, 4}  ·  hole at x = −4
⚠ A "hole" means the graph has a gap at that point — the function is undefined there but it is also not a zero.
f(x) = (4x² − 9) / (2x³ − x² − 4x + 3)
1
Factor numerator: 4x² − 9 = (2x−3)(2x+3) → candidates: x = 3/2 or x = −3/2
2
Factor denominator: 2x³ − x² − 4x + 3 = (2x−1)(x²−... try x=½: 2(⅛)−¼−2+3 = ¼−¼+1 = 1 ≠ 0. Try x=1: 2−1−4+3=0 ✔
3
Divide out (x−1): 2x³−x²−4x+3 = (x−1)(2x²+x−3) = (x−1)(2x+3)(x−1) = (x−1)²(2x+3)
4
x = −3/2: denominator = (−3/2−1)²(2(−3/2)+3) = (−5/2)²(0) = 0 → shared! hole at x = −3/2
5
x = 3/2: denominator = (3/2−1)²(2(3/2)+3) = (1/2)²(6) = 3/2 ≠ 0 ✔
Only zero: x = 3/2  ·  hole at x = −3/2  ·  poles at x = 1

4. Sign Chart (Tekenverloop)

The sign chart shows where f(x) is positive (+), negative (−), zero (0), or undefined (∅).

Method — 4 steps
1
Factor numerator and denominator completely
2
List all zeros (numerator = 0) and poles (denominator = 0) on a number line
3
Determine the sign of each factor in each interval (pick a test point)
4
Combine signs: (−)(−) = +, (+)(−) = −, etc.
f(x) = −(x²−9)(x−2)²(3x+1) / (x²+3x) — find sign chart

2
Critical points: x = −3, x = −⅓, x = 0 (pole), x = 2, x = 3

Note: (x+3) cancels → hole at x = −3 (not in domain)

5. Rational Inequalities

Solve by building a sign chart, then reading off the solution set.

(x³ − 1) / (4x³ + 4x² + x) ≤ 0
1
Factor numerator: x³ − 1 = (x−1)(x²+x+1) — note x²+x+1 > 0 always (discriminant < 0)
2
Factor denominator: 4x³+4x²+x = x(4x²+4x+1) = x(2x+1)²
3
Critical points: x = 0 (pole, double in denominator), x = −½ (pole, squared — sign doesn't flip!), x = 1 (zero)
4
Sign chart for (x−1) / (x(2x+1)²):
xx<−½x=−½−½<x<0x=00<x<1x=1x>1
f(x)−∅−∅−0+
Solution: (−∞, −½) ∪ (−½, 0) ∪ (0, 1]  =  ℝ \ {−½, 0} ∩ (−∞, 1]
⚠ At x = 0 and x = −½ the function is undefined → excluded. At x = 1 it equals 0 → included (≤).

Part 2 — Homographic Functions

Definition

f(x) = (ax + b) / (cx + d) with c ≠ 0 and ad ≠ bc
Why c ≠ 0?
If c = 0, the formula becomes (ax+b)/d — a simple linear function, not rational.
Why ad ≠ bc?
If ad = bc, e.g. f(x) = (2x+4)/(3x+6) = 2/3 — just a constant (horizontal line), not interesting.

The name comes from Greek: "homo-graphic" = same graph shape.
All homographic functions produce the same shape: a hyperbola.

Basic Form: f(x) = 1/x   (a=1, b=0, c=1, d=0)

Asymptotes — 3 Types

An asymptote is a line the graph approaches but never reaches.

Vertical Asymptote (VA)
x = a
Where denominator = 0 (and numerator ≠ 0)

f(x) → ±∞ as x → a
Horizontal Asymptote (HA)
y = a/c
Limit as x → ±∞
For f(x) = (ax+b)/(cx+d):
lim f(x) = a/c
Oblique Asymptote
y = mx + n
Only when degree of numerator = degree of denominator + 1
Not present in homographic functions
Quick formulas for f(x) = (ax+b)/(cx+d):
Vertical asymptote: set cx + d = 0 → x = −d/c
Horizontal asymptote: limx→±∞ f(x) = a/c
Domain: ℝ \ {−d/c}
Range (beeld): ℝ \ {a/c}
